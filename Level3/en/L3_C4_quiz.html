Question 1: NPV Interpretation

If a project has an NPV of $125,000, what does this indicate?

  • The project will cost $125,000
  • The project will create $125,000 in value above the initial investment
  • The project has a 12.5% return rate
  • The project will pay back in 1.25 years
Correct Answer: The project will create $125,000 in value above the initial investment. NPV represents net value creation after recovering the initial investment.

Question 2: IRR Decision Rule

A project has an IRR of 15% and your company's cost of capital is 10%. What should you conclude?

  • The project should be rejected
  • The project should be accepted
  • More information is needed
  • The project is exactly at breakeven
Correct Answer: The project should be accepted. When IRR exceeds the cost of capital, the project creates value.

Question 4: Indicator Classification

Which of the following is a LEADING indicator of ROI success?

  • Customer satisfaction improvement
  • User adoption rate of new processes

Question 8: Sensitivity Analysis

In sensitivity analysis, if a 10% reduction in benefits causes NPV to drop by 25%, this indicates:

  • The project is very robust to benefit variations
  • The project is highly sensitive to benefit assumptions
  • The project should definitely be approved
  • The discount rate is too high
Correct Answer: The project is highly sensitive to benefit assumptions. A 2.5x impact ratio indicates high sensitivity and risk.
